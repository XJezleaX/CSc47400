1D Time Series Graph¶

What is the data and how is it the data type? The data is the number of students registered across the years 2010 to 2020, measured each year. This is an example of a 1D time series graph because we have only one independent variable, years, and one dependent variable, the number of students registered.¶

2D Scalar Field Graph¶

What is the data and how is it the data type? The data is the equation cos(x)*cos(y), with an x, y, and z axis showing various values for each variable and the answer based on those values equaling the variables. I know this is a 2D scalar field graph because every (x,y) value in the space is assigned to only one scalar value on the z-axis.¶

2D Vector Field Graph¶

What is the data and how is it the data type? This figure shows a 2D vector field where each arrow represents the data vector <𝑥^2 − 𝑦^2 − 4, 2 𝑥 𝑦> at a point on the plane. We know this is a 2D vector field graph because each point in the two-dimensional space is assigned a vector with both magnitude and direction, as shown by the arrow's lengths and orientations.¶

3D Scalar Field Graph¶

What is the data and how is it the data type? The figure above shows a 3D scalar field representing the equation S(x y, z) = 1 - e^(-(x-10)^2/3)e^(-(y-10)^2)/2)e^(-(z-10)^2)/5) with color given the field value. It is a 3D scalar field because Because it assigns a single value to every point in the 3D space.¶
